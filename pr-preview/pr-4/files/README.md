# Static Files Placeholder

Place downloadable files here:

- **`resume.pdf`** — your CV/resume PDF for direct download

## Updating your CV PDF

1. Export your LaTeX resume as `resume.pdf`
2. Copy it to this folder: `static/files/resume.pdf`
3. Commit and push to trigger a redeploy

The "Download CV" button in the navbar and hero section links to `/files/resume.pdf`.
